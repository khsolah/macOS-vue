# MacOS Vue

This open-source project aims to reproduce MacOS on web.

Preview: https://khsolah-mac-os-vue.vercel.app/

## Stack

- Nuxt 3
- Typescript
- UnoCSS, SCSS
- Pinia
- Vitest
